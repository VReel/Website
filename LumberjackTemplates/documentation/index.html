Files structure

In version 3.0 we have integrated MailChimp subscription form. For your comfort, we've prepared two folders: AJAX form (demos with AJAX + PHP based form) and MailChimp form (demos include jQuery MailChimp form, more info here).

We do our best while making all files for Red Cap. Every file (HTML, CSS, JS) that was created by us has clear code, wrapped with comments - all this for your convenience.

Each demo has similar file structure:

  • css - CSS styles folder
  • fonts - webfont icons folder
  • img - folder for images
  • js - JavaScript files folder
  • php - PHP subscription script folder
  • video - local video files folder
  • index.html - main HTML file

CSS

For your comfort, we have prepared a lot of CSS classes, which are prefixed with lj- (for example: lj-text-center). Main CSS file is style.css which has been divided into 8 sections:

  • Import section - place where you include other CSS files (webfonts, your own stylesheets)
  • Basic styles - basic template styles and classes
  • Typography - styles responsible for typography look
  • Breakpoint for high-res devices - here you can put styles for high DPI devices (for example Retina)
  • Header - all styles for Header (top module in template)
  • Main - all styles for middle module (icon boxes) in template
  • Footer - all styles for Footer (bottom section in template)
  • Breakpoints - additional media queries for responsive CSS styles

All JavaScripts files are placed in js folder and have to be included at the bottom of the HTML file (before </body> tag).

<!-- jQuery -->

Fonts

Red Cap uses Font Awesome (webfont icon library) and Google Webfonts.

If you prefer static image icon over animation preloader, place your image please in img folder, then head over to css folder and edit style.css file.   background: #fff url('../img/preloader.gif') no-repeat center center;
  .

  .

By editing background property you can change background color.

In case you want to turn off preloader, please edit index.html and remove code shown below:

<!-- PRELOADER -->
<div class="lj-preloader"></div>
<!-- /PRELOADER -->

Then open lj-redcap.js in js folder and remove code shown below:

// preloader
$(window).load(function() {
  $(".lj-preloader").delay(100).fadeOut(500);
});

Scroll Reveal

Available since ver. 2.0 (demos 9-12)

Scroll Reveal script allows you to animate elements when they appear on screen. We strongly suggest you visiting authors page for detailed documentation: Scroll Reveal.

By adding a data-scroll-reveal attribute to an element, it will automatically be revealed (using default values) as soon as the element is within the viewport:

<!-- Reveal using defaults. -->
<div data-scroll-reveal> Hello world! </div>

Define your own reveal animation parameters, which you can do using natural, declarative language:

<a href="#" title="" data-scroll-reveal="enter left and move 40px over 1s">

All available video parameters are described on jquery.mb.YTPlayer script website: here

JavaScript files that have to be included:

  • jquery-1.11.0.min.js
  • device.min.js
  • jquery.mb.YTPlayer.js

JavaScript calls for YouTube video and controls are placed in lj-redcap.js file:

// JQUERY.MB.YTPLAYER
  $(function(){
    $(".yt-player").mb_YTPlayer();
  });

  // yt controls
  $('#yt-play').click(function(event){
    event.preventDefault();
    if ($(this).hasClass("fa-play") ) {
        $('.yt-player').playYTP();
    } else {
        $('.yt-player').pauseYTP(); 
    }
    $(this).toggleClass("fa-play fa-pause");
    return false;
  });
  $('#yt-volume').click(function(event){
    event.preventDefault();
    $('.yt-player').toggleVolume();
    $(this).toggleClass("fa-volume-off fa-volume-up");
    return false;
  });

lj-redcap.js must be included after other JS files.

Single image

Proper use of single image background can be seen in demo-5, demo-6 and demo-7. You can base your own website appearance on this demo files.

To achieve responsive image as header's background, you need to include following JS files:

  • jquery-1.11.0.min.js
  • jquery.backstretch.min.js

JavaScript call is placed in lj-redcap.js file:

// backstretch
$("header").backstretch("img/bg.jpg");

lj-redcap.js must be included after other JS files.

As a result of placing above code, file bg.jpg will be attached to <header>.

Image slideshow

Proper use of image slideshow can be seen in demo-1. You can base your own website appearance on this demo files.

To achieve responsive image slideshow as header's background, you need to include following JS files:

  • jquery-1.11.0.min.js
  • jquery.backstretch.min.js

JavaScript call is placed in lj-redcap.js file:

// backstretch
$("header").backstretch([
  "img/bg.jpg"
, "img/bg-2.jpg"
, "img/bg-3.jpg"
], {duration: 3000, fade: 750});

lj-redcap.js must be included after other JS files.

There are 2 paramaters available: duration (is the amount of time in between slides, in miliseconds) and fade (is value that determines how quickly the next image will fade in, in miliseconds).

As a result of placing above code, files bg.jpg bg-2.jpg bg-3.jpg will be attached to <header> and shown as slideshow.

Default logo image resolutions:

  • Retina (logo-2x.png) - width: 200px x height: 120px

In php folder you can find file called send.php. All PHP code for handling form submitting proccess can be put in this file. The variable that you can use in PHP functions is $_POST['subscription'] to get a value of entered e-mail address.

For those who are not familiar with PHP programming we've prepared ready-to-use basic examples.

- sending subscription address to your email:

// Please insert here your email address:
$myMail  = 'name@domain.com';

$to      = $myMail;
$subject = 'New subscription from '.$_POST['subscription'];
$message = $_POST['subscription'];
$headers = 'From: '.$_POST['subscription']. "\r\n" .
    'Reply-To: '.$_POST['subscription']. "\r\n" .
    'X-Mailer: PHP/' . phpversion();

mail($to, $subject, $message, $headers);

Please remember to change $myMail variable to your email address.

- storing subscriptions in file on your server:

$email = $_POST['subscription'];

$file = 'emails.txt';
// Open the file to get existing content
$current = file_get_contents($file);
// Append a new person to the file
$current .= $email."\n";
// Write the contents back to the file
file_put_contents($file, $current);

MailChimp form

Available since ver. 3.0

Red Cap has implemented jQuery MailChimp subscribe form with entered e-mail validation. Here is an entire HTML code:

<!-- FORM -->
<div class="row">
  <div class="col-xs-12 lj-form">
    <form id="newsletter">
      <div class="lj-block-center">
        <input type="text" value="" name="email" id="email" placeholder="Enter your e-mail">
        <input class="lj-text-uppercase" type="submit" value="Submit" name="submit">
      </div>
      <label for="email" class="lj-message"></label>
    </form>          
  </div>
</div>
<!-- /FORM -->

JavaScript files that have to be included:

  • jquery-1.11.0.min.js
  • jquery.ajaxchimp.js

All required script functions are included in our custom JS file lj-redcap.js. Here is a short code from JS:

// Ajax mailchimp
// Example MailChimp url: http://xxx.xxx.list-manage.com/subscribe/post?u=xxx&id=xxx
$('#newsletter').ajaxChimp({
  language: 'lj',
  url: 'put_your_mailchimp_url_here'
});

In url parameter you have to put your signup form URL. Follow these steps to get this URL address:

  • Login to MailChimp and go to Lists
  • Open your chosen list and click on Signup forms
  • Click on General forms
  • Find Signup form URL, copy the URL
  • Open in your browser this page: unshort.me, paste Signup form URL and press unshorten button
  • You'll resolve URL similar to this one:
    http://xxx.xxx.us8.list-manage1.com/subscribe?u=xxx&id=xxx
    Please add /post after subscribe:
    http://xxx.xxx.us8.list-manage1.com/subscribe/post?u=xxx&id=xxx
  • Finally, copy URL and paste to url in lj-redcap.js

A: Click on Downloads tab in your ThemeForest's profile page. There you will be able to rate our item. If you want to rate our item with 4 or less stars, please let us know why you do this. We just want to know about your opinion. This will help us to improve our items in future.

Q: Why images included in ThemeForest's .zip file are blurred?
A: The answer is really simple. We have no rights to sell them. They're used only for a preview on our demo previews. But don't worry! We have included in credits the URLs to all images/photos/videos in Credits section.

Q: My JavaScript script doesn't work, why?
A: Please check that you've included it properly, check filename in folder and HTML file. Also please make sure that you have properly call it in lj-redcap.js file.

A: Yes, you can. Don't forget to remove click function from lj-redcap.js (for more information, please head to Button's section).
